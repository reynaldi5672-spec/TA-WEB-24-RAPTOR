Portal resmi pariwisata Kota Bandar Lampung. Proyek ini dikembangkan sebagai portal informasi untuk destinasi wisata, kuliner, dan budaya di Bandar Lampung.

## Fitur Utama

- **Peta Destinasi Interaktif**: Eksplorasi tempat wisata dengan filter kategori, rating, dan pencarian cepat.
- **Sistem Favorit**: Simpan destinasi favorit Anda dengan integrasi `localStorage`.
- **Perbandingan Destinasi**: Bandingkan hingga 3 destinasi wisata sekaligus.
- **Mode Tema**: Dukungan antarmuka Gelap/Terang.
- **Responsif**: Desain yang dioptimalkan untuk perangkat mobile dan desktop.

## Teknologi

- **Framework**: Next.js 15 (App Router)
- **Bahasa**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL (Prisma ORM)
- **Komponen UI**: Lucide React Icons, SweetAlert2

## Pengembangan

### Prasyarat
- Node.js 18+
- npm/yarn/pnpm

### Menjalankan Server
```bash
npm install
npm run dev
```

## Fitur Unggulan 🚀

Website Wisata Bandar Lampung kini dilengkapi dengan alat perencanaan travel pintar:
1. **Peta Destinasi Interaktif**: Integrasi dengan Google Maps yang tersinkronisasi otomatis untuk menunjukkan rute lokasi.
2. **Kalkulator Biaya Wisata**: Penghitung anggaran akurat dilengkapi diagram proporsi pengeluaran interaktif.
3. **Checklist Bawaan Pintar**: Daftar barang bawaan (packing list) yang bisa diunduh secara offline dalam format `.txt`.
4. **Kuis Rekomendasi Destinasi**: Cari destinasi ideal Anda berdasarkan kuis kesukaan dan bagikan rekomendasinya secara instan.
5. **Widget Cuaca Real-Time**: Simulasi pemantau perkiraan cuaca adaptif di Kota Bandar Lampung.
6. **Pop-up Notifikasi & Konfirmasi Premium**: Integrasi SweetAlert2 untuk pengalaman notifikasi glassmorphic yang modern.
7. **Pencarian Cepat & Saran Otomatis**: Dilengkapi panel rekomendasi pencarian populer.
